#Full Stack Notes Application
            
## Description
This project is a full-stack Notes application built with Node.js, Express, and JavaScript.
It lets users create, view, edit, and delete notes through a browser interface. The frontend sends API requests to an Express backend, which handles CRUD operations and stores notes persistently.
